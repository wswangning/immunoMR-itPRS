# immunoMR-itPRS
Complete code for immune toxicity PRS &amp; MR analysis. Integrates ML clustering &amp; PRS-CS with two-sample, multivariable, mediation MR to assess immune-genetic lung cancer risk. Includes: data prep, immune subtyping, causal inference, colocalization, visualization. From "Immunotoxicity Phenotypes &amp; Health Outcomes" project. Reproducible.
